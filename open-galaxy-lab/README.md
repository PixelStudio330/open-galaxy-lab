# Open Galaxy Lab

A premium Next.js landing experience for Open Galaxy Lab.

## Structure

- `app/` — App Router pages and placeholders
- `components/sections/` — Modular landing page section components
- `lib/` — Motion presets and content data
- `styles/` — Global Tailwind CSS styles
- `app/admin` — Future admin placeholder route
- `app/api` — Placeholder API route

## Tech stack

- Next.js App Router
- TypeScript
- React
- Tailwind CSS
- Framer Motion
- Lucide React

## Commands

- `npm install`
- `npm run dev`
- `npm run build`
- `npm start`
